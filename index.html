Sustratos

Sustrato para orquideas

Ofreciendo la medida precisa para permitir el paso de aire mientras se retiene humedad.
La mezcla se encuentra libre de taninos y sustancias contaminantes, gracias a un proceso de lavado intensivo. Tiene el pH optimo para favorecer el adecuado desarrollo de las orquídeas epifitas, siendo perfecto para la siembra en maceta

Sustrato para cactus

La mezcla se encuentra enriquecida con calcio y magnesio, aportando minerales fundamentales para el desarrollo de la planta, ademas posee bajo contenido de materia organica y de nitrogeno disponible , con lo cual se logra el equilibrio de nutrientes especial para cactus y suculentas.

Arlita, Arcilla expandida

Este material proporciona buena aireacion a las raices, incremento del drenaje y control de la c onductividad.
Puede ser usado directamente como sustrato para hidroponia o en cultivo de plantas epifitas, tambien colocandolo en el fondo de las macetas para mejorar el drenaje.
